# Framework for NLP Pipeline

## Setup instructions
1. Clone this repo:
  ```shell
  >>> git clone https://github.com/SurajDonthi/nlp-models.git
  ```
2. Setup the conda environment with:
  ```shell
  >>> conda env create -f environment.yml
  ```
